# Machine-Learning

Repository for Machine-Learning course as part of ATU Computer Science - Data Analytics degree.

## About

This repository contains coursework, projects, and exercises for the Machine Learning module at Atlantic Technological University (ATU), part of the Computer Science - Data Analytics degree program.

## Course Content

This repository will include:
- Machine Learning algorithms and implementations
- Data analysis and preprocessing
- Model training and evaluation
- Course assignments and projects
- Jupyter notebooks with demonstrations

## Getting Started

### Prerequisites

- Python 3.x
- Jupyter Notebook
- Required Python libraries (to be added as needed)

## Structure

The repository will be organized as the course progresses with relevant folders for different topics and assignments.
